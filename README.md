### 기획 의도
저희 마미(마켓미림)은 회원가입, 로그인을 통해 물품을 구매할 수 있으며, 기본적으로 회원가입을 하게 되면 바로 이용을 할 수 있습니다. 학생들은 자기의 예산에 맞춰 간식들을 구매하여 먹을 수 있습니다. :chocolate_bar::apple::baby_bottle::cake:
